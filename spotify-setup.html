Spotify 連携設定

Spotify Developer Dashboard に Redirect URI を登録してから、ダッシュボードで連携してください。

Spotify Premium が必要です

このウィジェットの再生・一時停止・曲送り・曲戻し機能は、Spotify Premium プランに加入しているアカウントで利用してください。

直近の Spotify エラー

登録する Redirect URI

下記の URI を省略・変更せず、そのまま Spotify 側へ登録してください。

末尾の /spotify、大文字・小文字、末尾スラッシュを含めて完全一致させる必要があります。

設定手順

  1. Spotify Premium プランに加入しているアカウントを用意します。
  2. Spotify Developer Dashboard を開き、ログインします。
  3. この拡張機能で使用する Spotify アプリを開き、アプリの設定画面へ進みます。
  4. 「Redirect URIs」に上記の URI を追加します。
  5. Spotify 側の設定を保存します。
  6. Chrome の拡張機能ページで Nexus Dash を再読み込みし、新しいタブを開き直します。
  7. Nexus Dash の Spotify ウィジェットで「連携する」を押します。

エラーが出る場合

  • redirect_uri: Not matching configuration: Spotify 側へ登録した URI が上記の値と完全一致しているか確認してください。
  • 認証画面を開けない: Nexus Dash を Chrome 拡張機能として読み込んでいるか確認してください。
  • 再生操作が失敗する: Spotify Premium でログインし、Spotify アプリ側で再生デバイスをアクティブにしてください。
