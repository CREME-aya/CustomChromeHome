# Custom Chrome Homepage Dashboard

Google Discoverライクなデザインと「ガラスモーフィズム (Glassmorphism)」を取り入れた、高機能なブラウザ用カスタムホームページです。
Chromeの「新しいタブ」画面や、ローカルのスタートページとして使用することを想定しています。
HTML, CSS, Vanilla JavaScript だけで構築されており、外部ライブラリへの依存がありません。

## ✨ 主な機能

### 1. 美しいガラスモーフィズムUI
- 好きな壁紙画像を背景に設定可能（設定画面からURLを指定）。
- すりガラスのような半透明パネルで統一された、見やすく美しいデザイン。
- スマートフォンやタブレットでも崩れないレスポンシブデザイン。

### 2. 🌤 高精度な天気・気象ウィジェット
- **Open-Meteo API (気象庁シームレスモデル)** を利用し、正確な天気・気温・降水確率を表示。
- 設定サイドバーから「市区町村名をテキストで検索」するか、「現在地から取得」ボタンでリアルタイムの現在地周辺の天気を自動設定。
- 今後6時間分の「1時間ごとの降水確率」を美しいグラフで表示。

### 3. 🤖 マルチAI 同時検索＆チャット
- 1つの入力欄から、**ChatGPT (OpenAI)**, **Claude (Anthropic)**, **Gemini (Google)** の3つのAIに対して同時に質問が可能。
- 各AIの回答が専用のパネルに並列で表示されます。
- パネルごとに表示・非表示を切り替え可能。
- ※利用するには、各AIの「APIキー」を設定メニューから入力する必要があります。キーはブラウザのローカルストレージにのみ保存されます。

### 4. 📝 ToDoリスト ウィジェット
- 日々のタスクをサクッと追加・完了・削除できるシンプルなタスクリスト。
- タスクはブラウザに自動保存されるため、画面を閉じても消えません。

### 5. 🎵 Spotify ミニプレイヤー
- お気に入りのプレイリストやアルバムのURLを設定画面に入力するだけで、ダッシュボード上にミニプレイヤーが出現。
- ブラウザ上でSpotifyにログインしていれば、別タブを開かずにフル尺での音楽再生が可能です。

### 6. 🔗 クイックリンク機能
- 頻繁にアクセスするウェブサイトをショートカットとして登録可能。
- アイコン付きで表示され、ワンクリックでアクセスできます。

### 7. 📰 Discover風 記事フィード
- ニュースやブログ記事などをGoogle Discoverのようにカード形式で一覧表示。
- 気になる記事のお気に入り（★）登録、検索ボックスを使った絞り込みが可能。
- 記事をクリックすると画面遷移せず、ポップアップ（モーダル）でプレビューを表示します。

### 8. ⚙️ ウィジェット表示設定
- サイドバー（左上のハンバーガーメニュー）から、不要なウィジェットを自由に非表示（ON/OFF）にできます。
- 自分好みのシンプルな画面にカスタマイズ可能です。

## 🚀 使い方

1. **ファイルの準備**
   本プロジェクトフォルダ（`index.html`, `style.css`, `script.js` が入ったフォルダ）をPCの任意の場所に保存します。
2. **ブラウザで開く**
   Google Chrome等のブラウザで `index.html` を直接開いてください。
3. **拡張機能として使う場合**
   Chrome拡張機能として「新しいタブ」に設定したい場合は、別途 `manifest.json` を作成し、ブラウザの「拡張機能を読み込む」からフォルダを指定することでカスタムの新規タブとして利用できます。
4. **設定のカスタマイズ**
   画面左上の「☰」アイコンをクリックして設定メニューを開き、壁紙やAPIキー、天気エリア、SpotifyのURLなどを設定してください。

## 🛠 技術スタック
- **HTML5**
- **CSS3** (Flexbox, Grid, Glassmorphism, CSS Variables)
- **Vanilla JavaScript** (ES6+, Fetch API, LocalStorage, Geolocation API)

## 📡 利用している外部API
- **Open-Meteo API** (天気データ・降水確率)
- **Open-Meteo Geocoding API** (都市名の緯度経度変換)
- **BigDataCloud API** (無料・登録不要の逆ジオコーディング)
- **OpenAI API** (ChatGPT)
- **Anthropic API** (Claude)
- **Google Gemini API**
- **Spotify Web Embed**
